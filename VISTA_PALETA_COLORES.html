💻 Cómo Usar en tu CSS

Variables Disponibles

/* Fondos */ background-color: var(--bg-primary); background-color: var(--bg-secondary); background: var(--gradient-bg); /* Gradiente adaptado */ /* Texto */ color: var(--text-primary); /* Texto principal */ color: var(--text-secondary); /* Texto secundario */ color: var(--text-muted); /* Texto deshabilitado */ /* Colores */ color: var(--primary); /* Rojo/Primario */ color: var(--accent); /* Azul/Accent */ /* Efectos */ border: 1px solid var(--border-color); box-shadow: var(--shadow-sm); /* Sombra chica */ box-shadow: var(--shadow); /* Sombra normal */ box-shadow: var(--shadow-lg); /* Sombra grande */ /* RGBA Dinámico */ background: rgba(var(--primary-rgb), 0.1); box-shadow: 0 5px 15px rgba(var(--primary-rgb), 0.3);

Ejemplo de Uso

.card { background-color: var(--bg-secondary); color: var(--text-primary); border: 1px solid var(--border-color); box-shadow: var(--shadow); padding: 1.5rem; border-radius: 8px; } .card:hover { border-color: var(--primary); box-shadow: 0 10px 30px rgba(var(--primary-rgb), 0.2); }
